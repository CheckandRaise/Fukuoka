# 🚆 交通安排建议

## 推荐交通方式：JR + Kyushu Odan Bus / 租车 1 天

| 日期 | 交通 |
| --- | --- |
| **Oct 1–2** | 福冈市区地铁 / 公交 / 出租车 |
| **Oct 3** | 福冈 → 由布院：JR 特急或高速巴士 |
| **Oct 4** | 由布院 → 阿苏 → 黑川：推荐租车；不租车可查 Kyushu Odan Bus |
| **Oct 5** | 黑川 → 熊本：Kyushu Odan Bus 或租车 |
| **Oct 7** | 熊本 → 博多：九州新干线 / JR |

Kyushu Odan Bus 连接 **别府、由布院、黑川温泉、阿苏、熊本**，因此不租车也能完成这条线，只是阿苏停留时间会更受班次限制。

# JR Pass 是否需要买？

如果你主要坐 JR，可以考虑 **Northern Kyushu JR Pass**。

| Pass | 成人价格 |
| --- | ---: |
| **3 日** | ¥15,000 |
| **5 日** | ¥17,000 |

但这版行程中有一段 **由布院 / 黑川 / 阿苏 / 熊本** 更依赖巴士或租车，所以 JR Pass 是否划算要看你最后是否坐很多 JR。

大致判断：

- 如果只坐 **福冈 → 由布院** 和 **熊本 → 福冈**：不一定需要买 JR Pass
- 如果还加入别府、小仓、长崎等 JR 长距离移动：再考虑买 JR Pass

# 🌟 调整后的核心体验

**福冈美食 → 由布院温泉小镇 → 阿苏火山草原 → 黑川温泉乡 → 熊本城市游 → 福冈返程**

相比上一版，这版有三个好处：

1. **温泉从 3 晚缩减到 2 晚**，不会审美疲劳；
2. **熊本增加到 2 晚**，交通更稳，行程更松；
3. **保留由布院 + 黑川**，温泉质量没有明显下降。
