# 🚆 交通安排建议

## 推荐交通方式：JR + Kyushu Odan Bus / 租车 1 天

| **Oct 1–2** | 福冈市区地铁 / 公交 / 出租车 |

# JR Pass 是否需要买？

如果你主要坐 JR，可以考虑 **Northern Kyushu JR Pass**。JR 九州官网显示 Northern Kyushu Pass 覆盖福冈、大分、熊本、长崎、佐贺等北九州区域，价格为 **3 日 ¥15,000 / 5 日 ¥17,000**；适用普通列车、特急列车、九州新干线等，但 **博多—小仓之间的新干线不适用**。 [oai_citation:2‡JR Kyushu](https://www.jrkyushu.co.jp/english/railpass/railpass.html?utm_source=chatgpt.com)

| Pass | 成人价格 |
| --- | ---: |
| **3 日** | ¥15,000 |
| **5 日** | ¥17,000 |

这版行程的 JR 重点是：

- 博多 → 别府
- 熊本 → 博多

中间 **别府 → 黑川 → 熊本** 更依赖巴士或租车。因此是否买 JR Pass 要看你是否还会增加小仓、门司港、长崎等 JR 长距离移动。

大致判断：

- 如果只坐 **博多 → 别府** 和 **熊本 → 博多**：不一定需要买 JR Pass
- 如果还加入 **小仓 / 门司港 / 长崎 / 佐贺** 等 JR 长距离移动：再考虑买 JR Pass

# 🌟 调整后的核心体验

**福冈美食 → 别府温泉城市 → 阿苏火山草原 → 黑川温泉乡 → 熊本城市游 → 福冈返程**

相比由布院版本，这版更适合你现在的偏好：

1. **福冈到别府可直接坐 JR 特急**，交通更清晰；
2. **删除由布院**，减少小镇温泉重复感；
3. **温泉仍保留 2 晚**：别府偏城市温泉，黑川偏山间温泉乡；
4. **熊本住 2 晚**，节奏更稳，也方便返程。
